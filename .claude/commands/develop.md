# 自主开发模式

进入 Phase 2: 代码开发（自主模式）

## 前置检查

1. 确认 `solution/app_spec.txt` 存在
2. 确认 MCP 工具可用：`puppeteer`

## 任务

按照 `src/CLAUDE.md` 指南执行：

1. 阅读 `solution/app_spec.txt`
2. 生成或更新 `src/feature_list.json`
3. 启动前后端服务
4. 回归验证已完成功能
5. 选择下一个待开发功能
6. 实现 → Puppeteer 验证 → 更新进度 → 提交代码
7. 循环直到全部完成或会话结束

## 开始

读取规格文件和当前进度，继续开发。
